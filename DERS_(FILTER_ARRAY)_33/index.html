Filter Metodu

Bir Array metodu olan filter(), dizi elemanlarını döngüye alıp, istediğimiz koşula sahip olanlarla yeni bir dizi oluşturmak için kullanılır. Aşağıda products dizisinden, harf sayısı 5'ten fazla olanları almak istiyorum.

const products = ["Mikrofon","Kablo","Telefon","Bilgisayar","Mouse","Klavye","Ekran"]

const newProducts = products.filter(eleman => eleman.length > 5)

console.log(newProducts); //["Mikrofon", "Telefon", "Bilgisayar", "Klavye"]
